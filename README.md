# HolyWorld AutoReply Mod

Мод для автоматических ответов игрокам во время проверок на сервере HolyWorld.

## Функции

- Автоматически отвечает игрокам с префиксом `[CHECK]`
- Распознает более 50 категорий сообщений
- Адаптивные ответы в зависимости от контекста
- Трекинг состояния для каждого игрока
- Защита от флуда (cooldown 2 секунды)
- Задержка ответа 0.5-1.5 сек для естественности

## Команды

- `/ai start` - включить автоответчик
- `/ai stop` - выключить автоответчик
- `/ai status` - проверить статус

## Сборка

1. Поместите `gradle-wrapper.jar` в `gradle/wrapper/`
2. Запустите `./gradlew build`
3. JAR файл будет в `build/libs/`

## GitHub Actions

Мод автоматически компилируется при пуше в main/master.
Скачайте JAR из артефактов сборки.

## Категории ответов

| Категория | Описание |
|-----------|----------|
| insult_ban | Оскорбления → сигнал бана |
| confession_explicit | Явное признание в читах |
| leave_threat | Угрозы ливнуть |
| explicit_refusal | Явный отказ |
| code_submit | Отправка кода AnyDesk/RuDesk |
| discord_offer | Предложение Discord |
| greeting | Приветствие |
| reason_ask | Вопрос о причине проверки |
| claim_clean | Утверждение "я не читер" |
| what_is_anydesk | Вопросы об AnyDesk |
| downloading | Статус скачивания |
| cant_download | Проблемы со скачиванием |
| time_left | Сколько осталось времени |
| confession_question | Вопросы о признании |
| from_russia | Из РФ (проблемы с AnyDesk) |
| phone_player | Игрок с телефона |
| ... и другие | |
